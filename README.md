The Community Lost & Found Records System (CLFRS) is a user-friendly application designed to help schools, barangays, and local communities efficiently manage lost and found items.

This system solves the common problem of missing or unclaimed items by providing a centralized digital record where users can:

Log found items with photos or videos

Search the item database

Track claim status

Request to claim items

Improve transparency and accessibility

The project features a modern user interface, easy navigation, and local storage support (no external API needed).
